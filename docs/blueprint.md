# **App Name**: ConsensusFlow

## Core Features:

- Decision Submission: Decision submission page with fields for title, background and type of decision sought: Approve, Endorse, Note.
- Secretariat Dashboard: Secretariat dashboard displaying submitted decisions with title, status, and review button.
- Decision Review: Decision review page displaying decision details with an 'Approve for Meeting' action.
- Meeting Agenda: Meeting page displaying an agenda of scheduled decisions.
- Proposal Summarization: AI tool: Automated summarization of lengthy proposal backgrounds to streamline decision-making.
- Fitness to Review Question Generation: AI tool: Generation of questions designed to help determine a proposal's fitness to be reviewed, for use during proposal vetting.
- Decision Outcome: An outcome assigned to each decision once it has been considered. Includes: Approved, Endorsed, Noted, Not Approved.

## Style Guidelines:

- Background: Light Blue-Gray (#ECEFF1) for a calm and professional environment.
- Primary: Deep Blue (#3F51B5) for main interactive elements and headings.
- Accent: Soft Indigo (#5C6BC0) to highlight important actions and status indicators.
- Font: 'Inter', a grotesque-style sans-serif, for both headlines and body text to ensure readability and a modern feel.
- Minimalist icons representing decision statuses (e.g., pending, approved, rejected) and actions (e.g., submit, review, approve) to provide clear visual cues.
- Grid-based layout for organizing decision information and ensuring clarity on all pages.
- Subtle transitions and animations when submitting decisions, changing statuses, and navigating between pages to guide users through the workflow.
